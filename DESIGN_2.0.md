# 🎨 HabitDuel 2.0 — Дизайн и Новые Функции

## 📋 Обзор Нововведений

HabitDuel 2.0 включает масштабные улучшения дизайна и новые функции для улучшения пользовательского опыта.

---

## 🆕 Новые Экраны

### 1. **Достижения (Achievements)**
**Путь:** `/achievements`

- **Дерево достижений** по категориям:
  - 🏆 Общие
  - 🔥 Серии
  - ⚔️ Дуэли
  - 👥 Социальные
  - 🎉 События

- **Функции:**
  - Прогресс-бары для каждого достижения
  - Секретные достижения (скрыты до разблокировки)
  - XP награды за разблокировку
  - Фильтрация (все/разблокированы/заблокированы)

### 2. **Статистика (Stats)**
**Путь:** `/stats`

- **Вкладка "Обзор":**
  - Общая статистика (дуэли, победы, поражения, win rate)
  - Текущая и лучшая серия
  - Любимое время для чекинов

- **Вкладка "Активность":**
  - Тепловая карта активности (365 дней)
  - Мини-карта за 4 недели
  - Графики по неделям/месяцам

- **Вкладка "Соперники":**
  - Статистика противостояний (H2H)
  - Win rate против каждого соперника

### 3. **Магазин (Shop)**
**Путь:** `/shop`

- **Категории:**
  - 🎭 Аватары
  - 🎨 Темы оформления
  - ⚡ Бустеры
  - ✨ Эффекты

- **Валюта:** XP (⚡), Гемы (💎), Монеты (🪙)

- **Бустеры:**
  - Двойной XP (30 мин)
  - Заморозка серии
  - Дополнительная жизнь
  - Фора (+3 к серии)

## 🎨 Улучшения Дизайна

### Карточки Дуэлей
- **Градиентные фоны** в зависимости от статуса
- **Progress bars** для визуализации прогресса серии
- **Цветовое кодирование:**
  - Активные — оранжевый градиент
  - Открытые — синий градиент
  - Завершённые — зелёный градиент

### Анимации
- **Tween-анимации** при загрузке карточек
- **Shimmer-скелетоны** (планируется)
- **Confetti-эффект** при победе (планируется)

### XP Progress Ring
- Анимированное кольцо прогресса
- Отображение текущего уровня
- Визуализация прогресса до следующего уровня

### Тепловая Карта
- GitHub-style contribution graph
- Интерактивные ячейки с тултипами
- Цветовая индикация интенсивности

## 🎮 Геймификация

### Система Уровней
- **Формула:** каждые 200 XP = +1 уровень
- **Названия уровней:**
  - 1: Новичок
  - 2: Упорный
  - 3: Ветеран
  - 4: Легенда
  - 5+: Чемпион

### Начисление XP
| Событие | XP |
|---------|-----|
| Check-in | +10 |
| Streak Bonus (7 дней) | +5 |
| Победа в дуэли | +50 |
| Топ-3 в группе | +30 |
| Первая дуэль | +20 |
| Достижение | +25 |
| Ежедневный квест | +15 |

### Ежедневные Квесты
- Отметиться 1 раз
- Отметиться 3 раза
- Выиграть дуэль
- Серия 7 дней
- Дуэль с другом

- **Сезонные** — месячные события
- **Турниры** — на выбывание
