# 🌊 HabitDuel — Полная Дизайн-спецификация: Premium Glass

**Концепция:** Премиальный минимализм с эффектом глубины и стекла (glassmorphism). Спокойная концентрация вместо агрессии. Дуэли как партнёрство в росте — ощущение качественного продукта, а не аркадной игры.

**Платформы:** Android, iOS (mobile-first, вертикальная ориентация).  
**Технологии:** Flutter 3.x, Dart Shelf, PostgreSQL 16, WebSocket.

---

## 1. Философия дизайна

### Эмоциональный тон

- Спокойствие и ясность.
- Ощущение премиальности и заботы о пользователе.
- Мотивация через позитивное подкрепление, а не через страх проиграть.
- Прогресс важнее победы.

### Ключевые принципы

- **Один экран — один фокус.** На каждом экране один главный CTA, без конкуренции действий.
- **Глубина через слои.** Blur, opacity и z-уровни создают ощущение пространства.
- **Состояния всегда очевидны.** Загрузка, ошибка, пусто, успех — у каждого свой визуальный паттерн.
- **Адаптивная тема.** Светлая для дня, тёмная для вечера (авто + ручной выбор).
- **Минимум текста.** Цвет, иконка, позиция говорят больше слов.

### 2.7 Glass Effect

```dart
// Flutter реализация Glass-карточки
Container(
  decoration: BoxDecoration(
    color: isDark
      ? Color(0x991C1917)    // Night: 60% opacity
      : Color(0xB3FFFFFF),   // Day:   70% opacity
    borderRadius: BorderRadius.circular(16),
    border: Border.all(
      color: isDark
        ? Color(0x0DFFFFFF)  // Night: 5% white border
        : Color(0x33FFFFFF), // Day:  20% white border
      width: 1,
    ),
    boxShadow: [/* shadow из таблицы выше */],
  ),
  child: ClipRRect(
    borderRadius: BorderRadius.circular(16),
    child: BackdropFilter(
      filter: ImageFilter.blur(sigmaX: 20, sigmaY: 20),
      child: child,
    ),
  ),
)
```

## 4. Анимации

### 4.1 Тайминги и кривые

| Действие              | Длительность | Curve                  |
| --------------------- | ------------ | ---------------------- |
| Page transition       | 400ms        | `easeOutCubic`         |
| Button press          | 150ms        | `easeOut`              |
| Card press            | 200ms        | `easeOut`              |
| Modal open            | 300ms        | `spring (damping 0.8)` |
| Modal close           | 200ms        | `easeIn`               |
| Progress bar fill     | 800ms        | `easeOutQuart`         |
| Streak badge pulse    | 2000ms       | `easeInOut` (∞ loop)   |
| Check-in success      | 600ms        | `spring`               |
| Number counter change | 400ms        | `easeOutExpo`          |
| WS notification slide | 250ms        | `fastOutSlowIn`        |
| Confetti              | 1500ms       | —                      |

### 4.2 Детали ключевых анимаций

**Page Transition (Slide + Fade):**

```text
Новый экран: translateX(100% → 0) + opacity(0 → 1)
Старый экран: translateX(0 → -30%) + opacity(1 → 0)
Duration: 400ms
```

**Button Press:**

```text
scale: 1.0 → 0.96 → 1.0
opacity: 1.0 → 0.85 → 1.0
Duration: 150ms
```

**Check-in Success:**

```text
1. Кнопка: scale 1.0 → 0.95 → 1.1 → 1.0 (spring, 400ms)
2. Иконка меняется: spinner → checkmark (crossfade, 200ms)
3. Запуск Lottie checkin_success.json (loop: false)
4. ProgressBar: плавно обновляет ширину (800ms)
```

**WebSocket Notification (Snackbar):**

```text
Вход:  translateY(100% → 0) + opacity(0 → 1), 250ms
Авто-выход через 4 сек: opacity(1 → 0), 200ms
```

**Victory/Defeat Overlay:**

```text
Overlay fade-in: opacity(0 → 1), 300ms
Lottie animation: loop false, auto-play
Dismiss: tap или auto через 5с
```

**Streak Badge Pulse (активный streak > 3):**

```text
scale: 1.0 → 1.08 → 1.0, easeInOut, 2s, infinite
```

### 6.4 Адаптивность

- **Мобильный (< 600dp):** стандартный layout.

- SafeArea обязательна для iOS notch / Android gesture bar.
- `resizeToAvoidBottomInset: true` везде, где есть текстовые поля.
